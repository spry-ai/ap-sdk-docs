    Authentication¶

    • API Key (APIKeyQuery)

      • Parameter Name: ML_API_KEY, in: query.

    • API Key (APIKeyHeader)

      • Parameter Name: ML_API_KEY, in: header.

    • API Key (APIKeyCookie)

      • Parameter Name: ML_API_KEY, in: cookie.
